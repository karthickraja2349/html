Project 1

1. ONLINE FOOD ORDER SYSTEM

This is a console-based application built using Java and MySQL for the backend. The system allows users to view the menu, place orders, and track their orders. It uses MySQL to store information such as food items, prices, and user orders.

Features:

  • User can view a list of available food items.
  • User can place an order by selecting food items and specifying quantity.
  • User can check the status of their order.

Technologies:

  • Java (Core Java for the application logic)
  • MySQL (Database for storing food menu and orders)
  • JDBC (Java Database Connectivity for interaction between Java and MySQL)

Project 2

2. Voxpopuli System

The "Voxpopuli System" is a responsive web application built using Java, MySQL, Tomcat, and JSP. It serves as an online platform for users to voice their opinions and vote on various topics, providing insights based on the collected data.

Features:

  • Users can register, log in, and participate in polls.
  • Real-time results are displayed based on user votes.
  • Admin panel for managing polls and viewing results.

Technologies:

  • Java (JSP for dynamic web pages)
  • MySQL (Database for storing user details, polls, and votes)
  • Tomcat (Web server for running the application)
